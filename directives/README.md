# Diretivas (SOPs)

Este diretório contém os Procedimentos Operacionais Padrão (SOPs) para o agente.
Cada arquivo `.md` aqui deve descrever um processo específico.

## Formato Sugerido
- **Objetivo**: O que deve ser alcançado.
- **Entradas**: Dados necessários.
- **Execução**: Quais scripts em `execution/` devem ser chamados.
- **Saídas**: O que o usuário deve receber.
